Datepicker(Md2Datepicker)

The datepicker component is for date selecton functionality.

md2-datepicker settings

  • [(ngModel)] - Date - (Default: null) - binds to date.
  • [minDate] - Date - (Default: null)(Optional) - oldest selectable date.
  • [maxDate] - Date - (Default: null)(Optional) - latest selectable date.
  • [showWeeks] - boolean - (Default: true)(Optional) - if false week numbers will be hidden.
